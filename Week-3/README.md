# Week-3 SQL Assignment

## About

This is my Week-3 assignment for the Celebal Technologies Data Engineering Internship.

In this assignment, I worked on the Superstore dataset and used SQL to perform different business analysis tasks. The main focus was to understand and implement Subqueries, CTEs, Window Functions, and JOINs.

---

## Tasks Performed

- Imported the Superstore dataset
- Created Customers, Orders and Products tables
- Inserted data using `SELECT DISTINCT`
- Used Subqueries for different filtering operations
- Used CTEs to calculate total sales
- Applied `ROW_NUMBER()` and `RANK()` for customer ranking
- Combined JOIN, CTE and Window Functions
- Solved business-related SQL queries
- Saved query results and screenshots
